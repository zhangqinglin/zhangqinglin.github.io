GitHub+hexo搭建个人博客

说明：本人搭建博客主要参考https://jingyan.baidu.com/article/fd8044faf2e8af5030137a64.html

1、下载git客户端

地址：https://git-for-windows.github.io/

安装教程：https://jingyan.baidu.com/article/90895e0fb3495f64ed6b0b50.html

2、注册GitHub账号

地址：https://github.com/。注意，最后GitHub的博客地址是https://“username”.github.io,username是你注册GitHub的username(后文所有username指代您注册出的username)。

注册好GitHub，新建一个repository，名称为username.github.io(据说是GitHub限制必须如此)。

3、安装node.js

地址：https://nodejs.org/en/

安装教程：https://jingyan.baidu.com/article/fd8044faf2e8af5030137a64.html

4、与参考博客不同之处

_config.yml文件拉到最下面，我填写的repository是https开头的，后面提交博客时，会要求您输入GitHub账户和密码，而填写参考博客中的“git@github.com:WX-JIN/WX-JIN.github.io.git”,会要求你配置参考博文中的秘钥。

deploy:
type: git
repository: https://github.com/zhangqinglin/zhangqinglin.github.io.git
branch: master

5、博客撰写工具

撰写工具我使用的是markdown pad 2（地址：http://markdownpad.com/）
